Shortest Job First Algorithm

SJF algorithm, or Shortest Job First algorithm, schedules processes based on their execution time, with the shortest job being executed first.


Dining Philosopher's Problem

Look/C-look Disk Scheduling Algorithm

The look/c-look algorithm is a disk scheduling algorithm that services requests by scanning the disk in a specific direction and only reverses direction once the end of the disk is reached.


FIFO Page Replacement Algorithm

FIFO page replacement algorithm replaces the oldest page in the memory first.
